# Voyager Docker Services

Docker Compose stack for the Voyager development server (192.168.9.2).

## Services

### Monitoring

| Container | Image | Port | Description |
|-----------|-------|------|-------------|
| `beszel-hub` | henrygd/beszel | 8090 | Central monitoring hub for all homelab servers. Web UI at `:8090`. |
| `beszel-agent` | henrygd/beszel-agent | — | Monitors local Voyager system via Unix socket (`/beszel_socket/beszel.sock`). |

### Management

| Container | Image | Port | Description |
|-----------|-------|------|-------------|
| `hawser` | ghcr.io/finsys/hawser | — | Dockhand edge agent. Connects outbound to Dockhand on Borg (`:3004`) so Dockhand can manage Voyager containers without inbound port exposure. |

## Setup

### Environment Variables

```bash
cp .env.example .env   # or copy from hawser/.env.example for hawser token
```

The root `.env` holds all runtime secrets:

| Variable | Source | Description |
|----------|--------|-------------|
| `BESZEL_AGENT_TOKEN` | Beszel web UI → Add System | Agent auth token |
| `BESZEL_AGENT_KEY` | Beszel web UI → Add System | SSH public key |
| `HAWSER_TOKEN` | Dockhand UI → Settings → Environments → Voyager (Dockhand) → Hawser | Edge agent token |

### Beszel First-Time Setup (two-stage)

The Beszel agent needs credentials generated by the hub *after* first deploy:

1. Deploy: `docker compose up -d`
2. Open `http://voyager:8090`, create admin account
3. Add System → Host/IP: `/beszel_socket/beszel.sock`
4. Copy TOKEN and KEY into `.env`
5. `docker compose restart beszel-agent`

### Hawser Token

The Hawser token is generated in the Dockhand UI on Borg:

1. Open Dockhand at `http://borg:3004`
2. Settings → Environments → Voyager (Dockhand) → Hawser
3. Generate token, copy into `.env` as `HAWSER_TOKEN`
4. `docker compose restart hawser`

The token is stored as an Argon2 hash in Dockhand's SQLite DB on Borg.

## Operations

```bash
# View all logs
docker compose logs -f

# Restart a service
docker compose restart hawser

# Update images
docker compose pull && docker compose up -d
```

## Notes

### Hawser (Dockhand agent)

- Uses edge mode: Hawser connects *outbound* to Dockhand — no ports exposed on Voyager
- Dockhand URL: `ws://192.168.9.7:3004/api/hawser/connect`
- Agent name in Dockhand: `Voyager (Dockhand)`
- Only mounts `docker.sock` — host CPU metrics in Dockhand will reflect Docker container aggregate, not total host CPU (use Beszel for accurate host metrics)

### Beszel Agent

- Uses Unix socket instead of TCP — no network port needed
- Mounts `/:/host:ro` for accurate host-level CPU, RAM, disk, and network metrics
- Hub is at `http://voyager:8090` and aggregates metrics from all homelab servers
